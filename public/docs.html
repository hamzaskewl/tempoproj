clippy

dashboard docs

Real-time Twitch stream intelligence. Detects chat spikes, classifies moments with AI, and auto-clips highlights.

Pay-per-use via MPP. No API keys needed.

How it works

1
Firehose ingests chat from Twitch channels in real-time via websocket. Only watched channels get full tracking.
2
Per-channel rate tracking (5s burst window, 30s baseline). Spike threshold scales with channel size — big streamers need less % jump.
3
LLM On watched channels, Claude classifies the mood and describes what happened.
4
Twitch Every spike on a watched channel auto-creates a Twitch clip. The LLM description becomes the clip title.
5
MPP All paid endpoints use Tempo MPP — agents pay with USDC, no API keys.

For agents

Any agent with a Tempo wallet can use clippy. No API keys, no signup.

Install tempo CLI

curl -fsSL https://tempo.xyz/install.sh | sh
tempo wallet login

Browse what's hot

# Free — no payment needed
tempo request https://stream-intel-production.up.railway.app/trending

Get channel stats (paid)

# Pays $0.001 automatically via MPP
tempo request -X POST \
  -H "Content-Type: application/json" \
  -d '{"channel": "xqc"}' \
  https://stream-intel-production.up.railway.app/channel

Watch for spikes (session)

# Opens payment channel, $0.03 per spike
# LLM classifies mood + auto-clips
# Auto-closes when stream ends, unused deposit refunded
tempo request -X POST \
  https://stream-intel-production.up.railway.app/watch/jynxzi

From the website

No agent needed. Use clippy directly from the browser.

1. Watch the dashboard

The dashboard shows live spikes streaming in from all channels. Click any spike to see the chat snapshot and VOD link.

2. Add channels to watch

# Add a channel to auto-clip watchlist
POST /watch-clip/jynxzi

# See what you're watching
GET /watch-clip

# Remove a channel
DELETE /watch-clip/jynxzi

Watched channels get LLM-classified on every spike and auto-clip to Twitch.

3. Classify any moment

Click "classify with ai" on any spike in the dashboard, or hit the API:

GET /moments/890/classify

# Returns:
{
  "mood": "shock",
  "description": "Streamer made a terrible decision that chat tried to stop",
  "clipWorthy": true
}

4. Connect Twitch for auto-clipping

Visit /auth/twitch to authorize clip creation. Once connected, watched channels auto-clip every spike with the AI description as the clip title.

Payment flow

Clippy uses Tempo MPP for payments. Two models:

Charge (one-off)

Agent sends request → gets 402 → pays USDC → gets response. One transaction per request.

Session (streaming)

Agent deposits USDC into escrow → $0.03 deducted per spike via off-chain voucher → unused deposit refunded when session closes. Auto-closes when stream goes offline.

Stack

Twitch chat firehose → Node.js/Express → mppx (Tempo MPP) → Claude (mood classification) → Twitch Helix API (auto-clip)

Deployed on Railway. Built at Tempo HQ, SF — MPP Hackathon (Tempo x Stripe).
